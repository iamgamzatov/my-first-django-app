# My First Django Application 🚀

Веб-приложение, созданное на фреймворке Django. Проект демонстрирует освоение базовой архитектуры Django, работу с шаблонизатором и подключение статических файлов.

### На странице реализовано:
* Интеграция кастомных шаблонов (Templates).
* Настройка и подключение статических файлов (Стилей CSS, фоновых изображений) через блок `staticfiles`.
* Понимание архитектуры MVT (Model-View-Template).

### Демонстрация интерфейса:
На главной странице отображается приветственный интерфейс ("Привет от Расула Гамзатова") с фоновым изображением и интерактивной кнопкой перехода на следующие этапы разработки.

### Как запустить локально:
1. Клонировать репозиторий.
2. Установить Django: `pip install django`.
3. Запустить сервер: `python manage.py runserver`.
4. Открыть в браузере: `http://127.0.0.1:8000/`.
